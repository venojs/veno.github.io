Veno基于Vite, 所以也需要 Node.js 版本 >= 12.0.0

开始新项目 #

使用npm

npm create veno

使用yarn

yarn create veno

完整语法

yarn create veno my-veno-app --vue3

提示

  • 模板分为vue2和vue3, 都是单页应用形态, 如果需要多页应用, 需要少量配置

从Vue Cli项目迁移 #

  1. 安装veno
yarn add @veno/kit
  1. package.json添加script
"script": {
    "dev": "veno dev",
    "build": "veno build",
}
  1. 把index.html从public目录迁移到根目录

that's all~ 如果需要更多的自定义, 可以使用配置文件
