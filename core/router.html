文件路由 #

Veno默认选择src/pages下所有的.vue文件来生成路由配置, 但不包括以下文件：

  • 已.或_开头的文件和文件夹
  • component或者components下的所有文件

路由规则 #

路由规则与vite-plugin-pages类似

基本路由 #

  • src/pages/users.vue -> /users
  • src/pages/users/settings.vue -> /users/settings
  • src/pages/users/index.vue -> /users

动态路由 #

已中括号开始的文件名会生成动态路由

  • src/pages/users/[id].vue -> /users/:id
  • src/pages/[user]/settings.vue -> /:user/settings

嵌套路由 #

目录下拥有layout.vue的, 会生成嵌套路由

src/pages/
  ├── users/
  │  ├── layout.vue
  │  ├── index.vue
  │  └── [id].vue

[{
    path: '/users',
    component: '/src/pages/layout.vue',
    children: [
        {
            path: '',
            component: '/src/pages/users/index.vue',
            name: 'users'
        },
        {
            path: ':id',
            component: '/src/pages/users/[id].vue',
            name: 'users-id'
        }
    ],
}]

提示

  • 所有路由均为动态导入
